📁 Pembuat Folder Multi-Level + Range

💡 Pilih dari dropdown, atau ketik manual path folder yang sudah ada.
🎯 Fitur Range Generator: Gunakan format Nama{start:end} untuk membuat banyak folder otomatis.
  • Pertemuan-{1:16} → Pertemuan-1, Pertemuan-2, ..., Pertemuan-16
  • Minggu-{01:12} → Minggu-01, Minggu-02, ..., Minggu-12 (dengan nol depan)
  • Bab{1:5} → Bab1, Bab2, ..., Bab5
  • Pertemuan-{1:16}\Materi → Pertemuan-1\Materi, Pertemuan-2\Materi, ... (bertingkat)
📝 Format lain: Pisahkan dengan koma atau Enter. Gunakan \ untuk folder bertingkat.
